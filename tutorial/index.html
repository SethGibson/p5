processing.js and dat.gui tutorial

made with processing.js and dat.gui

    Parametric functions with some organic-ness.  Orgametrics?  Paraganics?  This started out as a planetary motion study, but it reminded me more of some swarming sims I'd done a while back, so Bugs! it was.  With this sketch, it's fun to just let it run and watch the behavior cycle and evolve.

    Left drag to change the sim bounds, right-click to reset the sim.
